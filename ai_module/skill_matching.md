# Skill Matching Module

## What is Skill Matching?

Skill Matching is the process of comparing a student's resume skills with the required skills of a selected career.

## Process

1. Read Resume
2. Extract Skills
3. Read career_skills.csv
4. Compare Skills
5. Find Missing Skills
6. Generate Report

## Example

Resume Skills:
- Python
- SQL
- Git

Career:
AI Engineer

Required Skills:
- Python
- Machine Learning
- Statistics
- Git

Matched Skills:
- Python
- Git

Missing Skills:
- Machine Learning
- Statistics
